This is the code for Bundle segmentation as multiple Linear Assignment Problem (LAP).

See Sharmin et al., 'White Matter Tract Segmentation as Multiple 
Linear Assignment Problems', Fronts. Neurosci., 2017.

- The main file is lap_multiple_examples.py

- Cython code for linear assignment can be compiled with:
   python setup_lapjv.py build_ext --inplace
